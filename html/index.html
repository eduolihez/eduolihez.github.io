Estructura Básica

Una página HTML comienza con una estructura básica que incluye:

<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <title>Título de la Página</title>
    </head>
    <body>
        Contenido de la página.
    </body>
</html>

Historia del HTML

HTML ha evolucionado desde su creación para adaptarse a las necesidades modernas. Aquí un resumen:

  • 1990: Creación de HTML por Tim Berners-Lee.
  • 1995: Publicación de HTML 2.0.
  • 1997: HTML 4.0 introduce mejoras importantes.
  • 2000: XHTML combina HTML con XML.
  • 2011: HTML5 añade soporte para multimedia y APIs avanzadas.
  • 2017: HTML 5.2 establece la base moderna del lenguaje.

Etiquetas Comunes

  • <p>: Define un párrafo.
  • <h1> a <h6>: Define encabezados de diferentes niveles.
  • <a>: Crea enlaces.
  • <img>: Inserta imágenes.
  • <ul> y <ol>: Crea listas no ordenadas y ordenadas.

Elementos Multimedia

HTML5 soporta audio y video de forma nativa:

Video

<video src="video.mp4" controls></video>

Audio

<audio src="audio.mp3" controls></audio>

Tablas

Las tablas se utilizan para organizar datos:

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>

Formularios

Los formularios recogen datos del usuario:

<form action="/submit" method="post">
    <label for="name">Nombre:</label>
    <input type="text" id="name" name="name" required>
    <input type="submit" value="Enviar">
</form>

Notas y Consejos

  • Utiliza <meta charset="UTF-8"> para garantizar compatibilidad de caracteres.
  • Valida tus páginas con herramientas como W3C Validator.
  • Consulta compatibilidad de navegadores en Can I Use.
